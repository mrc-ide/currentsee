# Package flow labels

A dataset with flow start and end codes and associated user-facing flow
label

## Usage

``` r
flow_labels
```

## Format

A data frame with 39 rows and 3 variables:

- flow_start:

  Character string describing the intervention package (e.g. `"cm"`,
  `"cm, itn"`) of flow origin.

- flow_end:

  Character string describing the intervention package (e.g. `"cm"`,
  `"cm, itn"`) of flow terminus

- flow_label:

  Character string of label to display for flow.

## Source

M3CPI project dataset
