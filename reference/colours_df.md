# Package colour palette

A dataset mapping malaria intervention package combinations to assigned
colour codes for consistent plotting and visualisation across outputs.

## Usage

``` r
colours_df
```

## Format

A data frame with 24 rows and 2 variables:

- node:

  Character string describing the intervention package (e.g. `"cm"`,
  `"cm, itn"`).

- colour:

  Hexadecimal colour code associated with the package.

## Source

M3CPI project dataset
